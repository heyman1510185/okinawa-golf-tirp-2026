# OKINAWA GOLF TRIP 2026

## 旅行用スケジュールマップ Webページ 設計書（Okinawa Soft UI）

------------------------------------------------------------------------

## 1. 目的

-   Google Sheetsよりスマホで見やすい旅行スケジュール閲覧を実現する
-   日別に時間順の予定を追える構造にする
-   各予定からGoogle Mapsをワンタップで開ける
-   "システムっぽさ"を抑えた、沖縄っぽい柔らかいトーンのUIを実装する

------------------------------------------------------------------------

## 2. 技術方針（Vercel前提）

### 推奨スタック

-   Next.js（App Router）+ TypeScript
-   Tailwind CSS
-   データはCSVをビルド時にJSON化して静的配信

### データ取り込み方針

-   `2026_OKINAWA_GOLF_TRIP - sequences.csv` を `data/`
    ディレクトリに配置
-   `scripts/convertCsvToJson.ts` を用意し、`npm run build` 前に変換
-   生成物：`data/trip.json`

理由： - 地図APIやDBを使わず、最短で閲覧可能な状態にする -
旅行中も軽量で安定動作する構成にする

------------------------------------------------------------------------

## 3. 情報設計（IA）

### 画面構成（1ページ完結）

-   `/` のみで完結するSPA構成
    -   上部：旅タイトル・日付レンジ・メンバー表示（任意）
    -   日別タブ（例：2/9〜2/15）
    -   フィルタ（カテゴリchips / エリアdropdown）
    -   タイムライン（カードUI）
    -   「地図で見る」セクション（エリア別リンク一覧）

将来的に `/day/2-09` など分割可能だが、初期は1ページ構成とする。

------------------------------------------------------------------------

## 4. データ仕様（CSV→JSON）

### 入力CSV列

-   `day`：例 `2/9`
-   `time`：例 `14:45`
-   `category`：例 `移動, 飲み, ゴルフ`
-   `content`：例 `ホテルチェックイン`
-   `notes`：補足情報
-   `area`：例 `那覇_西`
-   `google maps`：Google Maps URL

### 正規化後Event型

``` ts
type Event = {
  id: string;
  day: string;
  time?: string;
  category?: string;
  title: string;
  notes?: string;
  area?: string;
  mapUrl?: string;
  sortKey: number;
}
```

### 欠損データ処理ルール

-   `day` が空の行は無視する
-   `time` が空の場合も表示可能（終日扱い）
-   `mapUrl` が空の場合は地図ボタンを非表示

## 5. UI/UX詳細（スマホ最適）

### 5.1 ヘッダー

-   タイトル：OKINAWA GOLF TRIP 2026
-   サブタイトル：日付レンジ（CSVから自動算出可能）
-   右上：Shareボタン（Web Share API対応）

### 5.2 日別タブ

-   横スクロール可能なpill型タブ
-   選択中は背景色変更 + 下部ドット表示
-   ページ遷移なしで即時切替

### 5.3 フィルタ

-   カテゴリ：複数選択可能chips
-   エリア：dropdown選択
-   「地図ありのみ」トグル

### 5.4 タイムライン

-   時間順カード縦並び
-   左に細いタイムラインガイド線
-   カード構成：
    -   上段：time + category badge
    -   中段：title
    -   下段：notes（あれば）
    -   右下：地図ボタン（mapUrlあれば）

contentに「1軒目\_」形式が含まれる場合は分解表示。

### 5.5 地図で見るセクション

-   日単位でareaをグルーピング
-   各エリア見出し下にmapUrlリンク一覧表示

## 6. デザイン仕様（沖縄ソフトUI）

### カラートーン

-   海：深いブルー（メイン）
-   空：淡いブルー（背景グラデーション）
-   砂：サンドベージュ（カード背景）
-   珊瑚：コーラル（アクセント）

### 質感設計

-   背景：グラデーション + 微細ノイズ
-   角丸：20〜24px
-   影：弱く広がるシャドウ
-   アイコン：細線系（lucide-react想定）

### タイポグラフィ

-   英数字：Inter / Manrope
-   日本語：Noto Sans JP / Zen Kaku Gothic New
-   タイトルのみserif混在可（可読性優先）
